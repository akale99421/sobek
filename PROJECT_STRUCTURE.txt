SERVICES (when running)
=======================

Local Development (docker-compose):
  - frontend:3000      → React app with hot-reload
  - backend:8000       → FastAPI with auto-reload
  - postgres:5432      → PostgreSQL database
  - qdrant:6333        → Vector database

Production (GKE):
  - frontend (2 pods)  → Nginx serving React
  - backend (2 pods)   → FastAPI application
  - postgres (1 pod)   → StatefulSet with persistent storage
  - qdrant (1 pod)     → StatefulSet with persistent storage
  - ingress            → HTTPS load balancer
